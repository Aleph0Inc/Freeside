NAME

FS::dbdef_unique.pm - Index object

SYNOPSIS

  use FS::dbdef_index;


    # see FS::dbdef_colgroup methods

DESCRIPTION

FS::dbdef_unique objects represent the (non-unique) indices of a table (FS::dbdef_table). FS::dbdef_unique inherits from FS::dbdef_colgroup.
